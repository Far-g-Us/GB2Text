# Аудит проекта GB2Text

## 📊 Обзор проекта

**GB2Text** - это инструмент для извлечения и перевода текста из ROM-файлов игровых приставок Game Boy, Game Boy Color и Game Boy Advance.

### Основные характеристики:
- **Язык**: Python 3.11+
- **GUI**: Tkinter с ttkbootstrap
- **Архитектура**: Модульная с системой плагинов
- **Локализация**: Поддержка 4 языков (EN/RU/JA/ZH)
- **Тестирование**: Полный coverage с pytest
- **Лицензия**: MIT
