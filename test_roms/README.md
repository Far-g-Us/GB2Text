# Тестовые ROM файлы

Эта папка предназначена для хранения ROM файлов игр Game Boy, Game Boy Color и Game Boy Advance, которые используются для тестирования.

## Инструкция

1. **Поместите ROM файлы** в эту папку
2. Поддерживаемые форматы:
   - `.gb` — Game Boy
   - `.gbc` — Game Boy Color  
   - `.gba` — Game Boy Advance

## Требования

- ROM файлы должны быть **легально приобретены** вами
- Файлы должны быть в рабочем состоянии (не повреждены)

## Автоматическое обнаружение

Тесты автоматически находят все `.gba`, `.gbc` и `.gb` файлы в этой папке и используют их для интеграционного тестирования.
