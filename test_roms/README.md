# Тестовые ROM файлы

В эту папку следует помещать ROM файлы для тестирования приложения GB2Text.

## Требования к файлам

- Файлы должны быть в формате `.gb`, `.gbc` или `.gba`
- Файлы должны быть легально принадлежать вам
- Не распространяйте чужие ROM файлы

## Рекомендуемые тестовые игры

Для полноценного тестирования рекомендуется использовать ROM следующих типов игр:

1. **Game Boy (GB)** - старые игры с однобайтной кодировкой
2. **Game Boy Color (GBC)** - игры с расширенной кодировкой
3. **Game Boy Advance (GBA)** - игры с более сложной структурой

## Примечание

Эта папка не отслеживается Git'ом. Вы можете добавить свои тестовые ROM файлы локально для отладки и тестирования.
