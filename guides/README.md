# Руководства по извлечению текста

Здесь хранятся пошаговые инструкции для извлечения текста из конкретных игр.

## Структура гайда

```json
{
  "game_id": "MY_GAME_01",
  "description": "Руководство по извлечению текста для My Game",
  "steps": [
    {
      "title": "Шаг 1: Подготовка",
      "description": "Описание действий"
    }
  ],
  "tips": [
    "Полезный совет"
  ]
}
```

## Поля

| Поле | Тип | Описание |
|------|-----|----------|
| `game_id` | string | ID игры (совпадает с именем файла) |
| `description` | string | Краткое описание |
| `steps` | array | Массив шагов |
| `tips` | array | Массив советов |

## Поля шага

| Поле | Тип | Описание |
|------|-----|----------|
| `title` | string | Название шага |
| `description` | string | Описание действий |

## Как создать гайд

1. Создайте JSON-файл с именем `{game_id}.json` (например, `POKEMON_RED.json`)
2. Используйте [`example_homebrew.json`](/guides/example_homebrew.json) как образец
3. Опишите шаги извлечения текста для этой игры
4. Добавьте полезные советы

## Оценивание гайдов

Вы можете оценить гайд от 1 до 5 звёзд. Оценки хранятся в файлах `{game_id}.rating`.

Оценка влияет на порядок отображения гайдов в приложении.
