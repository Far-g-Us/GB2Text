# Предложения по улучшению GB2Text

## ✅ Реализовано

1. ✅ **Валидация ROM** - добавлена функция `validate_rom_file()` в core/rom.py
   - Проверка расширения (.gb, .gbc, .gba)
   - Проверка минимального размера (32KB)
   - Проверка максимального размера (64MB)
   
2. ✅ **Константы UI** - добавлены в gui/main_window.py
   - COMBOBOX_WIDTH = 30
   - COMBOBOX_WIDTH_SMALL = 15
   - DEFAULT_PADDING = 10
   - и др.

3. ✅ **Улучшенное определение сегментов с ML** - добавлен модуль core/ml_classifier.py
   - Использует scikit-learn (RandomForestClassifier) для классификации
   - Fallback на эвристический анализ при отсутствии sklearn
   - Покрытие тестами: 94%

4. ✅ **Поддержка нескольких таблиц символов** - добавлен модуль core/multi_charmap.py
   - Классы CharTable и MultiCharmapSegment для работы с несколькими charmap
   - Автоматическое определение таблиц символов в данных
   - Разбиение сегментов по таблицам при декодировании
   - Покрытие тестами: 100%

5. ✅ **Улучшенное определение нестандартных кодировок** - добавлен класс EncodingDetector
   - Функция analyze_custom_encoding() для анализа кодировки
   - Обучение детектора на примерах (learn_encoding)
   - Автоматическое определение Shift-JIS последовательностей
   - Предложение оптимальной таблицы символов (suggest_charmap)
   - Интеграция с CharMapDecoder через MultiCharMapDecoder

6. ✅ **Шаблоны конфигураций** - добавлен шаблон plugins/config/_template_gb_basic.json
   - Базовый шаблон для игр Game Boy с English текстом
   - Конфигурация charmaps, segments, validation
   - Пример для создания собственных конфигураций

7. ✅ **API для создания плагинов** - добавлен модуль core/plugin_api.py
   - Расширенный базовый класс ExtendedGamePlugin
   - Система хуков (HookManager, HookType)
   - Фабрика плагинов (PluginFactory)
   - Декоратор @register_plugin для регистрации
   - Поддержка жизненного цикла плагина
   - Покрытие тестами: 100%

8. ✅ **Валидация переводов** - добавлен модуль core/translation_validator.py
   - Классы ValidationError, ValidationResult
   - TranslationValidator для проверки длины текста
   - BatchTranslationValidator для пакетной проверки
   - Проверка управляющих символов и форматирования
   - Покрытие тестами: 98%

9. ✅ **Автоматическое заполнение нулевых переводов** - добавлен модуль core/translation_filler.py
   - TranslationFiller с различными стратегиями (COPY_ORIGINAL, PLACEHOLDER, TEMPORARY_MARK)
   - SmartTranslationFiller с умным определением типов сегментов
   - Batch заполнение и декоратор @auto_fill_empty
   - Покрытие тестами: 92%

10. ✅ **Шаблоны для GBC и GBA** - добавлены шаблоны в plugins/config/
    - _template_gbc.json для Game Boy Color
    - _template_gba.json для Game Boy Advance
    - Расширенные настройки pointer_base, bank_switching, compression
